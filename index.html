Ode-simulation-gsl-template

Template C++ Code for Simulating Ordinary Differential Equations using GSL

Download .zip Download .tar.gz View on GitHub

This repository contains template C++ Code for simulating ordinary differential equations using the Gnu Scientific Library (GSL), which can be found here: http://www.gnu.org/software/gsl/

The essentials.h header file contains a data structure called struct parameters. However, for any long-term development, a parameters class should be used so that you can manipulate the parameters with member functions.
